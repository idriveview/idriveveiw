Сделайте свою работу в облачном хранилище более комфортной с программой IDriveView!

Смотреть на YouTube
Это видео является обзором программы, которая позволяет пользоваться облаком почти как обычной папкой на компьютере. Облачные хранилища S3 предназначены для профессионального использования и поэтому имеют высокую скорость доступа. Программа не сохраняет файлы на компьютере, а работает напрямую с облаком. Она способна подключаться к S3-хранилищам, используя стандартные данные для такого подключения: Endpoint и два ключа. Я испытал эту программу на трёх облачных хранилищах: YandexCloud, IDrive и Selectel. Во втором ролике я подробно рассказываю, как зарегистрироваться в этих облачных сервисах и получить необходимые данные для подключения. Обязательно посмотрите — там я показываю эти сервисы изнутри и рассказываю о некоторых важных нюансах, чтобы всё работало корректно.

Мышь и горячие клавиши

  • keyboard Z + ЛКМ при клике на папку — открывает папку без превью картинок.
  • delete Delete — удаляет выбранный элемент.
  • keyboard_arrow_left keyboard_arrow_right Стрелки влево и вправо — переход к следующей картинке (в окне просмотра картинок).
  • space_bar Пробел — останавливает и запускает видео (в окне просмотра видео).
  • save Ctrl + S — сохраняет изменения в текстовом файле.
  • select_all Ctrl + A — выделяет все объекты папки.
  • mouse Можно мышью выделить несколько объектов.
  • control_point_duplicate Ctrl + ЛКМ — выделяет элемент, если он не был выделен, и снимает выделение, если был.
  • swap_vert Shift + ЛКМ — выделяет элемент, если нет других выделений; если есть — заполняет пространство между выделениями.
  • menu ПКМ — открывает контекстное меню: "Скачать на ПК", "Переименовать", "Удалить", "Снять выделение", "Скопировать имя в буфер обмена", "Получить ссылку на файл", "Закрыть меню".

Поддерживаемые форматы

  • description Просмотр и редактирование текстовых файлов: txt, json, cs, html, css, js, jsx, py, idv
  • image Программа поддерживает просмотр картинок формата: jpg, jpeg, png, webp
  • movie Можно пробовать просматривать видео формата: mp4, mpeg, mkv, avi, mov, wmv, flv, webm
  • audiotrack Аудио: mp3

Примечание

  • warning Переименование папки с большим количеством вложений может занимать довольно продолжительное время.
  • cloud_queue Если облако сильно заполнено, то подсчёт занятого пространства может занять продолжительное время.
  • folder_open Программа поддерживает длинные имена файлов, но не поддерживает длинные имена папок. (Длинное имя — это примерно более 100 байт, то есть 100 английских символов или 50 русских).
  • description Файл логов и файл конфигураций находятся в папке "IDriveVeiw" в "Documents": "C:\Users\igorNik\Documents\IDriveView"
  • autorenew Если хотите сбросить всех пользователей и настройки программы, удалите папку: "C:\Users\igorNik\Documents\IDriveView"
  • desktop_windows Программа тестировалась на Windows 10 и 11
